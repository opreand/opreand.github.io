Working with relational selectors

This page is meant to help you practice using relational selectors. There are four main types of relational selectors

Four Relational Selectors

  • Descendant
  • Adjacent Sibiling
  • Child
  • General Sibline

Relational selectors work differently than more common selectors used in CSS.

Common CSS Selectors

There are three common types of selectors used with CSS.

  • HTML elements
  • ID
  • Class

In addition to relational selectors and common selectors, there is one last type of selector, the All Elements selector. The All Elements selector applies a style to all HTML elements in a webpage and it uses an asterisk.

Relational selectors make it easier to apply CSS to different HTML elements without having to write code for each element or include a large number of IDs and Classes.
